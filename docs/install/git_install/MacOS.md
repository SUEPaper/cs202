## Homebrew 安装 git

:::note

Homebrew 是啥？请看[Mac 必备神器 Homebrew](https://zhuanlan.zhihu.com/p/59805070)

:::

在安装好 Homebrew 后, 命令行中输入如下命令就可以安装 git。

```bash
brew install git
```

查看安装好后的 git 版本

```bash
git --version
```
