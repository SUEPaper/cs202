# 什么是 HTML
HTML（超文本标记语言）是一种用于创建网页结构和内容的标记语言。它是互联网上最基本和常用的标记语言之一。HTML使用一系列标（由尖选号选起的单词或符号）来确定义网页的结构和各种元素，如标题、段落、链接、图像和表格等。

HTML的主要作用是描述和确定网页的结构，它通过使用不同的标签来指示文本、图像、链接和其他媒体在网页中的显示方式和位置。标签可以包含属性，用于下一步确定义元素的特性和行为。

HTML是一种基于文本的标记语言，可以使用简单的文本编辑器创建和编辑HTML文件。当浏览器加载HTML文件时，它会解析HTML代码并将其转换为可视化的网页内容，呈现给用户。通过CSS（层次样式表）和JavaScript等其他技术，可以对HTML页面进行样式化和交互性的增强。
```bash
My cat is very grumpy
```
如果我们想要将这行文字单独呈现，可以将这行文字封装成一个段落（Paragraph）&#60p> 元素：
```bash
<p>My cat is very grumpy</p>
```
:::tip
备注： HTML 标签不区分大小写。也就是说，输入标签时既可以使用大写字母也可以使用小写字母。例如，标签 &#60title> 可以写作 &#60title>、&#60TITLE>、&#60Title>、&#60TiTlE> 等，也都可以正常工作。不过，从一致性、可读性来说，最好仅使用小写字母。
:::
## HTML 的历史与发展
HTML 代表超文本标记语言，是一种用于创建网页和应用程序的标准标记语言。以下是其历史和发展的概述：

1、起源（1980 年代）：HTML 的根源可以追溯到 1980 年代，当时英国计算机科学家 Tim Berners-Lee 提出了一种在研究人员之间共享信息的系统。这导致了超文本概念的发展和第一个 HTML 版本的创建。

2、HTML 1.0 (1991)：HTML 的第一个官方规范，称为 HTML 1.0，由 Tim Berners-Lee 于 1991 年发布。它提供了一种简单的标记语言，用于构建包含标题、段落和链接等基本元素的文档。

3、HTML 2.0 (1995)：HTML 2.0 作为 HTML 的改进版本于 1995 年推出。它添加了表单输入、图像映射和表格等新功能，支持更复杂的网页布局。

4、HTML 3.2 (1997)：HTML 3.2 是 HTML 发展过程中的一个重要里程碑。它引入了对背景图像、文本格式和框架集等附加功能的支持，从而允许创建更具视觉吸引力和交互性的网站。

5、HTML 4.01 (1999)：HTML 4.01 成为广泛采用的 HTML 版本。它增加了对样式表、脚本和多媒体元素（如音频和视频）的支持。它还为编写 HTML 代码引入了更严格的规则和指南。

6、XHTML (2000)：XHTML（可扩展超文本标记语言）是作为使用 XML 语法的 HTML 4.01 的重新表述而引入的。XHTML 1.0 遵循更严格的 XML 语法规则，使其与其他基于 XML 的技术更加兼容。XHTML 1.1 于 2001 年发布。

7、HTML5 (2014)：HTML5 是 HTML 的当前主要版本，代表了 Web 开发能力的重大飞跃。它于 2014 年作为万维网联盟 (W3C) 的候选推荐首次引入，并于 2014 年达到推荐状态。HTML5 引入了许多新功能，包括对多媒体元素的原生支持、用于绘制图形的画布、改进的表单控件以及用于更好地与 JavaScript 集成的 API。

8、持续发展：HTML5 不断发展，并定期添加新功能和增强功能。据我所知，截至 2021 年 9 月，最新的 HTML 工作草案是 HTML 5.3。W3C 和 WHATWG（Web 超文本应用技术工作组）监督 HTML 标准的持续开发和维护。

请务必注意，HTML 通常与其他技术结合使用，例如用于样式化的 CSS（层叠样式表）和用于向网页添加交互性的 JavaScript。这些技术的结合构成了现代 Web 开发的基础。
## HTML 与 CSS、JavaScript 的关系
HTML（超文本标记语言）、CSS（层叠样式表）和 JavaScript 是 Web 开发中使用的三种基本技术。他们一起工作来创建和增强网页。

HTML 是用于创建网页结构和内容的标准标记语言。它提供了一组预定义元素（标签），用于定义网页的不同组件，例如标题、段落、图像、链接、表单等。HTML 单独负责网页的基本结构和语义。

CSS 是一种样式语言，用于控制 HTML 元素在网页上的呈现和外观。它允许您定义颜色、字体、布局、间距和其他视觉方面。使用 CSS，您可以将表示层与内容层分开，从而更容易管理和更新网站的视觉设计。

JavaScript 是一种为网页添加交互性和动态行为的编程语言。它允许您创建交互式功能、执行计算、操作 HTML 和 CSS、处理用户交互以及与服务器通信。JavaScript 可用于创建动画、验证表单、实现滑块、构建交互式地图等等。它通过添加功能并使网站更具吸引力和响应速度来补充 HTML 和 CSS。

HTML、CSS 和 JavaScript 共同构成了现代 Web 开发的基础。HTML 提供结构和内容，CSS 添加样式和表示，而 JavaScript 添加交互性和动态行为以创建丰富的交互式 Web 体验。它们通常一起用于构建网站和 Web 应用程序。
## 什么是 HTML5

HTML5 是超文本标记语言 (HTML) 的第五次修订，它是在万维网上创建和构建内容的标准标记语言。它是用于设计和呈现网页和应用程序的核心技术。

开发 HTML5 是为了增强其前身的功能，使 Web 开发人员能够创建更具动态性和交互性的网站。它引入了新的功能和元素，使嵌入多媒体内容、处理图形和动画以及构建更复杂的 Web 应用程序变得更加容易。

HTML5 中引入的一些关键特性和改进包括：

语义元素：HTML5 引入了新的语义元素，如&#60header>, &#60nav>, 等，为网页内容提供更好的结构和意义&#60article>，&#60section>，&#60footer>

多媒体支持：HTML5 提供对嵌入多媒体内容的原生支持

## HTML5 的历史和发展
HTML5 是一种用于创建网页内容的标记语言，它是 HTML4 的升级版。以下是 HTML5 的历史和发展：

1、2004年：W3C 开始制定 HTML5 规范。

2、2008年：第一个 HTML5 草案发布。

3、2011年：HTML5 规范正式成为 W3C 推荐标准。

4、2014年：HTML5.1 发布。

5、2016年：HTML5.2 发布。

6、2022年：W3C 宣布对 HTML5 进行维护，并开始制定下一代 Web 标准。

HTML5 引入了许多新功能，如语义化标签、视频和音频标签、本地存储、Web Workers、Canvas 等。这些功能使得开发者能够更轻松地构建现代网站和应用程序，并提供了更好的用户体验。


## 浏览器开发者工具
浏览器开发者工具是设计用于帮助开发人员在 Web 浏览器中调试和分析网页的工具集合。以下是浏览器开发者工具通常提供的一些功能：

控制台(Console) - 允许开发人员查看 JavaScript 错误、警告，并与 JavaScript 执行环境进行交互。

元素检查器(Element Inspector) - 允许开发人员查看和修改 HTML 和 CSS，以及它们之间的关系。

网络监视器(Network Monitor) - 允许开发人员查看网站加载速度、HTTP 请求和响应、缓存控制等信息。

资源管理器(Resource Manager) - 允许开发人员查看和管理页面上的资源，例如图片、样式表和脚本文件。

性能分析器(Performance Analyzer) - 允许开发人员分析网页性能、识别瓶颈和优化机会。

应用程序调试(Application Debugging) - 允许开发人员调试在浏览器中运行的 Web 应用程序。

大多数现代浏览器都提供了内置的开发人员工具，例如 Chrome 的 DevTools、Firefox 的 Developer Edition、Safari 的 Web Inspector 等。这些工具可以极大地帮助开发人员加快开发进程、诊断问题并提高代码质量。
