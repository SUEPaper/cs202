# 表格

## 表格基础
HTML表格是网页中用于显示和组织数据的常识元素之一。以下是HTML表格的基础知识：
```bash
<table>
  <tr>
    <th>表头单元格1</th>
    <th>表头单元格2</th>
  </tr>
  <tr>
    <td>数据单元格1</td>
    <td>数据单元格2</td>
  </tr>
  <tr>
    <td>数据单元格3</td>
    <td>数据单元格4</td>
  </tr>
</table>
```
- &#60table>标签确定了整个表格的开始和结束。
- &#60tr>标定义了表示的行（行定义）。
- &#60th>标签确定了表格的表格头元格式，用于显示列表标题。
- &#60td>标签确定了表格的数据单位，用于显示现实的数据。

在上面的示例中，表格有两列和三行。第一行是表格头，包含了两个表格头单位格。接下来的两行是数据行，每行有两个数资料单元格。
## 表头、表身和表脚
在HTML表格中，可以将表格内容分为三个主要部分：表格头部（thead）、表格身体（tbody）和表格脚（tfoot）。这种结构可以提供高级表格的可读性，并允许在装扮中包含大量数据时保持表格的结构一致。以下是对每个部分的详细说明：
1、表头（thead）：表头部分用于确定表格的列标题或表格的描述性信息。通常，表头位于表格的顶部，并使用标签来定义表头单元格。可以使用标签定义表头&#60th>行&#60tr>。
2、表身（tbody）：表身部分包含实际的表格数据。数据行位于表身中，使用&#60tr>标签确定每行。每个数据行使用&#60td>标签来定义数元格。
3、表脚（tfoot）：表脚部分用于包含表格式的总结信息或脚注。通常，表脚位于表的底部，使用标签来确定权利。与表情和表情类似，可以使用和标记来&#60tfoot>定义&#60tr>表情&#60td>脚行和元格。
```bash
<table>
  <thead>
    <tr>
      <th>列1</th>
      <th>列2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>数据1</td>
      <td>数据2</td>
    </tr>
    <tr>
      <td>数据3</td>
      <td>数据4</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="2">总计：4 条数据</td>
    </tr>
  </tfoot>
</table>
```
在上面描述的例子中，&#60tfoot>包含一个表脚行，其中使用&#60td>来创建一个跨越列表的单位格式（colspan="2"）来显示总计的信息。

通过将表格规划分为表头、表身和表脚，可以提出高表格的可访问性和可维护性，并使其更易于理解和理解。
## 表格跨行和跨列
在HTML中，您可以使用rowspan属性来跨行合并单一格式，使用colspan属性来跨行合并单一格式。

要跨行并单一格，将rowspan属性添加到要合并的单一格中，并设置为要跨过的行数。例如，要将一个元格跨过两行，可以这样写：
```bash
<td rowspan="2">跨行单元格</td>
```
要跨列表并单一格，将colspan属性添加到要合并的单一格中，并设置为要跨更多的列表。可以这样写：
```bash
<td colspan="3">跨列单元格</td>
```
下面是一个演示示例，展示了如何同时跨行和跨列合并并元格：
```bash
<table>
  <tr>
    <td>单元格1</td>
    <td>单元格2</td>
    <td rowspan="2">跨行单元格</td>
  </tr>
  <tr>
    <td colspan="2">跨列单元格</td>
  </tr>
  <tr>
    <td>单元格4</td>
    <td>单元格5</td>
    <td>单元格6</td>
  </tr>
</table>
```
在上面的例子中，第一行有三个单位格，第一个单位格占一列，第二个单位格占列，第三个单位格超过两行。第二行有一个元格，跨越两列。第三行有三个元格。

注意，当单元跨行或跨列时，它们会影响其他单元的布局。因此，在设计表格时，请确保考虑到合并元格可能会引发的影响。
