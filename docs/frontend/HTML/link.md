# 超链接和锚点
超链接非常重要——它们使互联网成为一个互联的网络。

## 超链接标签（a）
超链接（Hyperlink）是一个指向其他网页、文档、图片或其他资源的链接。它由元素&#60a>表示，在元素的href属性中指定目标标准来源的网址（URL）。
:::tip
URL 可以指向 HTML 文件、文本文件、图像、文本文档、视频和音频文件以及可以在网络上保存的任何其他内容。如果浏览器不知道如何显示或处理文件，它会询问你是否要打开文件（需要选择合适的本地应用来打开或处理文件）或下载文件（以后处理它）。
:::
通过将文本或其他内容包裹在 &#60a> 元素内，并给它一个包含网址的 href 属性（也称为超文本引用或目标，它将包含一个网址）来创建一个基本链接。下面是一个超链接的演示：
```bash
<a href="http://www.example.com">这是一个超链接</a>
```
### 块级链接
就像之前提到的那样，任何内容，甚至块级内容都可以作为链接出现。如果想让标题元素变为链接，就把它包裹在锚点元素（&#60a>）内，像这个代码段一样：
```bash
<a href="https://developer.mozilla.org/zh-CN/">
  <h1>MDN Web 文档</h1>
</a>
<p>自从 2005 年起，就开始记载包括 CSS、HTML、JavaScript 等网络技术。</p>
```
### 图片链接
如果有需要作为链接的图片，使用 &#60a> 元素来包裹要引用图片的 &#60img> 元素。
```bash
<a href="https://developer.mozilla.org/zh-CN/">
  <img src="mdn_logo.svg" alt="MDN Web 文档主页" />
</a>
```

## href 属性
HTML中的href属性用于指定超链接（链接）的目标URL或资源的位置。

href属性是&#60a>元素的一个常见属性，用于创建超链接。它也可以在其他元素中使用，例如&#60link>、&#60script>和&#60img>等元素。

下面是一些href属性的使用示例：
1、在&#60a>元素中，href属性指定链接的目标URL：
```bash
<a href="http://www.example.com">这是一个超链接</a>
```
2、在&#60link>元素中，href属性指定外部样式表的URL：
```bash
<link rel="stylesheet" type="text/css" href="styles.css">
```
3、在&#60script>元素中，href属性指定外部JavaScript文件的URL：
```bash
<script src="script.js"></script>
```
4、在&#60img>元素中，href属性指定图像文件的URL：
```bash
<img src="image.jpg" alt="图片">
```
需要注意的是，不同的元素可能对href属性有不同的要求和语言义。例如，&#60a>元素的href属性用于指定链接目标，而&#60img>元素的src属性用于指定图像文件的URL。

除此之外，href属性还可以与锚点（锚链接）一起使用，通过在URL中指定锚点名称来创建内部页面导航。例如：
```bash
<a href="#section1">跳转到第一部分</a>
```
href属性是用于指定链接目标或资源位置的重要属性，它使我们有能力在 HTML 中创建丰富的链接和导航功能。
## 锚点的创建和使用
锚点（Anchor）是页面内部的导航点，用于在同一页中的不同位置之间进行导航。通过在超链接的属性中指定锐点名称称，并在目标位置使用仅用点名作标记，href可以在页面内部创建链接。

下面是一个使用锚点的示例：
```bash
<a href="#section1">跳转到第一部分</a>

...

<h2 id="section1">第一部分</h2>
<p>这是第一部分的内容。</p>
```
超链接的href属性使用#section1指定了一个锚点名称。在页面的其他位置，&#60h2>标签使用id属性设置了与锚点名称相同的价值，作为金点的标记。当用用户点击“跳转到第一部分"的链接时，页面会滚动到具有相关性的位置，即"第一部分"的标题处。

通过结合超链接和锁点的使用，可以实现在同一页面内的导航、跳转和书签功能。
