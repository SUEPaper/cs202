# 列表标签
现在让我们把注意力转向列表。在生活中，列表无处不在——从你的购物清单到你每天下意识地遵循的方向列表，再到你在这些教程中遵循的指令列表！在网络上，列表也无处不在，我们有三种不同的类型需要注意。
## 无序列表（ul）
无序列表（Unordered List）：使用&#60ul>标签包列表项，每个列表项使用&#60li>标签包。无序列表将使用特定符号号（如圆点、方块）作为项目编号。
```bash
<ul>
  <li>列表项1</li>
  <li>列表项2</li>
  <li>列表项3</li>
</ul>
```
## 有序列表（ol）
有顺序列表（Ordered List）：使用&#60ol>标签包列表项，每个列表项使用&#60li>标签包。有顺序列表将按顺序给列表项添加加数字或字母编号。
```bash
<ol>
  <li>列表项1</li>
  <li>列表项2</li>
  <li>列表项3</li>
</ol>
```
## 列表项（li）
在HTML中，可以使用&#60li>标签来创建列表项。&#60li>标签用于包列表中的每个项目。
```bash
<li>列表项1</li>
<li>列表项2</li>
<li>列表项3</li>
<li><a href="https://www.example.com">链接1</a></li>
<li><a href="https://www.example.com">链接2</a></li>
<li><a href="https://www.example.com">链接3</a></li>
```
每个列表项都将显示为一个链接。你可以根据需要根据具体的内容和设计要求来组织和制定列表项。
